# battery_ui_beta
battery info float ui for windows app.

## Getting Started

### Prerequisites
It need to `node 10.14.x` for run dev mode.

### Installing
Install node require modules...
```
npm i
```
### Run the project!
You can use run this project.`npm start` or `yackage start`

### Stop the project!
You can use `Ctrl + Q` for stop the project.
